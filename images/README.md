# Images Directory

Place your images in this directory.

## Required Images

For the demo to work properly, you need to add the following images:

1. `profile.jpg` - Your profile/avatar image (recommended size: 350x350px)
2. `project1.jpg` - Thumbnail for Project 1 (recommended size: 600x400px)
3. `project2.jpg` - Thumbnail for Project 2 (recommended size: 600x400px)
4. `project3.jpg` - Thumbnail for Project 3 (recommended size: 600x400px)

You can use any image format (jpg, png, webp) but make sure to update the references in the HTML accordingly.
