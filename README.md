# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. This website serves as a showcase for my professional skills, projects, and contact information.

## Features

- Responsive design that works on all devices
- Light and dark mode toggle with persistent settings
- Smooth scrolling and animations
- Project showcase with filterable categories
- Skills and experience section
- Contact form
- Modern UI with clean design

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (Vanilla JS, no frameworks)
- Font Awesome for icons
- Google Fonts

## Getting Started

To deploy this portfolio website to GitHub Pages:

1. Fork this repository
2. Enable GitHub Pages in your repository settings
3. Your website will be available at `https://yourusername.github.io`

## Customization

1. Replace the placeholder content in `index.html` with your own information
2. Update the profile image and project images in the `images` folder
3. Modify the color scheme by changing the CSS variables in `css/style.css`
4. Add your own projects to the projects section

## Contact Form

The contact form is set up for demonstration purposes. To make it functional, you'll need to:

1. Set up a form submission endpoint (e.g., using Formspree, Netlify Forms, or your own backend)
2. Update the form submission logic in `js/main.js`
